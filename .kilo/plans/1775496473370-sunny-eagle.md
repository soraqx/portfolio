# Game-Themed Portfolio Website Plan

## Project Overview
- **Project Name**: Game Portfolio
- **Type**: Single-page portfolio website
- **Core Functionality**: A game-themed, minimalistic portfolio with customizable color palette, dark/light mode, and interactive elements
- **Target Users**: Developers showcasing their work to potential employers/clients

## Technical Stack
- **Framework**: React 18 with Vite
- **Styling**: Tailwind CSS + custom CSS variables
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Default Accent**: #FF00FF (Magenta)

## UI/UX Specification

### Layout Structure
1. **Navigation** - Fixed top nav with logo, section links, theme toggle, color picker
2. **Hero Section** - Full viewport height, centered content with typing effect
3. **Tech Stack** - Grid of pill-shaped badges
4. **Projects** - 3-column card grid (responsive: 1 col mobile, 2 tablet, 3 desktop)
5. **GitHub Contributions** - Static heatmap (7x52 grid)
6. **Contact** - Social icons in a row
7. **Footer** - Minimal with year and tech credits

### Visual Design

#### Color Palette (Dark Mode Default)
- `--bg-primary`: #0A0A0A
- `--bg-secondary`: #111111
- `--bg-tertiary`: #1A1A1A
- `--text-primary`: #FFFFFF
- `--text-secondary`: #A0A0A0
- `--accent`: #FF00FF (default, user-changeable)
- `--accent-glow`: rgba(255, 0, 255, 0.3)

#### Color Palette (Light Mode)
- `--bg-primary`: #FAFAFA
- `--bg-secondary`: #F0F0F0
- `--bg-tertiary`: #E5E5E5
- `--text-primary`: #0A0A0A
- `--text-secondary`: #666666

#### Typography
- **Font Family**: "JetBrains Mono" for headings/accents, "Inter" for body
- **Headings**: H1: 4rem, H2: 2.5rem, H3: 1.5rem
- **Body**: 1rem, line-height 1.6

#### Spacing
- Section padding: 6rem vertical
- Container max-width: 1200px
- Card gap: 2rem

#### Visual Effects
- Custom cursor: 8px dot + 24px ring
- Glitch effect on hover (text-shadow offset)
- Scanline overlay (subtle CSS pseudo-element)
- Grid pattern background (low opacity)
- Card hover: scale(1.02) + glow shadow

### Components

#### Color Palette Controller
- Hue slider (0-360)
- Saturation slider (0-100%)
- Lightness slider (0-100%)
- Preview swatch
- Reset button
- Persists to localStorage

#### Theme Toggle
- Sun/Moon icon button
- Smooth transition (0.3s)
- Persists to localStorage

#### Custom Cursor
- Dot: 8px solid accent color
- Ring: 24px ring with accent color, 2px stroke
- Hover state: ring scales to 1.5x

#### Project Cards
- Image/icon placeholder (128x128)
- Title (H3)
- Description (2 lines max)
- Tech tags (pill badges)
- "Live Demo" and "GitHub" buttons

#### GitHub Heatmap
- 52 columns (weeks) x 7 rows (days)
- Color levels: 5 shades from bg-tertiary to accent
- Tooltip on hover (date + count)

## Functionality Specification

### Core Features
1. **Color Palette Control** - Real-time CSS variable updates
2. **Dark/Light Mode** - Toggle + system preference detection + localStorage
3. **Smooth Scroll** - Anchor links with scroll behavior
4. **Typing Effect** - Hero text typewriter animation
5. **Hover Animations** - Cards, buttons, links with Framer Motion

### User Interactions
- Click nav links в†’ smooth scroll to section
- Drag color sliders в†’ update accent color globally
- Toggle theme в†’ switch dark/light + save preference
- Hover project cards в†’ scale + glow effect
- Click social icons в†’ open link in new tab

### Data Handling
- All content in separate data files (easy to edit)
- localStorage for theme and accent color

### Edge Cases
- Color picker: clamp values to valid ranges
- Theme: fallback to dark if localStorage unavailable
- Mobile: hide custom cursor, use touch-friendly targets

## Acceptance Criteria
1. Color picker updates accent color in real-time across all elements
2. Dark/Light mode toggle works and persists across page reload
3. All sections render correctly on mobile (320px), tablet (768px), desktop (1200px+)
4. Keyboard navigation works (tab through links, enter to activate)
5. GitHub heatmap displays 52x7 grid with proper coloring
6. Animations are smooth (60fps) and don't cause layout shifts
7. All placeholder content clearly marked with comments
